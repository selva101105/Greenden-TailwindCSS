# Greenden Website

Greenden is a modern, responsive website for a plant and flower business offering artificial and natural plants. The website includes a home page, products page, and contact page with a mobile-friendly responsive design and interactive features.

## Features

- Responsive Navbar for both desktop and mobile view  
- Mobile Side Navbar with smooth sliding animation  
- Home page sections:
  - Header with hero section and call-to-action buttons
  - About Us section with features (Large Assortment, Free & Fast Shipping, 24/7 Support)
  - Best Seller section with interactive hover effects
  - Customer Reviews section
  - Newsletter signup with email validation  
- Products Page with search bar and product grid layout  
- Contact Page with contact form, input validation, and structured layout  
- Footer section for all pages  



## Technologies Used

- HTML5  
- Tailwind CSS 2.2.19  
- JavaScript  
- Font Awesome Icons

 Usage:
- Navigation: Use the navbar links to switch between Home, Products, and Contact pages.
- Mobile Menu: Click the hamburger icon to open the mobile menu and X icon to close.
- Newsletter: Enter a valid email in the newsletter section on the home page to be redirected to the Contact page.
- Contact Form: Fill in Name, Email, and Message fields and click Submit. Email validation is included.
- Product Page: Browse products and use the search bar to filter products.
